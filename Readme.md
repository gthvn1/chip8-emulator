# Chip8 Emulator

## What?
- A *chip8* emulator in rust
- [Changelog](https://github.com/gthvn1/chip8-emulator/blob/master/Changelog.md)
- To run it: `cargo run -- chip8-roms/IBM_logo.ch8 > /tmp/IBM_logo.mem`
